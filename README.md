# Conceit - Business Landing Page

Một trang đích (landing page) dạng corporate/business được thiết kế phản hồi (responsive) bằng HTML, CSS, JavaScript và thư viện Bootstrap. 

## Tính năng chính
- **Thanh điều hướng (Navbar)**: Thanh điều hướng cố định (sticky) tích hợp menu hỗ trợ di động.
- **Trình chiếu ảnh (Carousel/Slider)**: Khung trình chiếu hình ảnh tự động chuyển động mượt mà.
- **Hiệu ứng Fade-in**: Các thành phần (như phần About Us, Services) sẽ xuất hiện với hiệu ứng mượt mà (fade-in) khi cuộn trang.
- **Phong cách hiện đại**: Sử dụng FontAwesome cho các biểu tượng (icons) và cách bố trí khoa học, rõ ràng.

## Cài đặt và Sử dụng
1. Clone dự án về máy:
   ```bash
   git clone <địa_chỉ_repo_của_bạn>
   ```
2. Mở file `index.html` trong trình duyệt web để xem giao diện.
3. Chỉnh sửa nội dung, CSS hoặc JS trong các thư mục tương ứng.

## Cấu trúc thư mục
- `css/`: Chứa các tệp CSS định dạng giao diện tĩnh và Bootstrap.
- `js/`: Chứa các tập lệnh JavaScript, bao gồm Bootstrap và jQuery.
- `images/`: Chứa các hình ảnh hiển thị trên trang web.
- `Fonts/`: Thư mục chứa các phông chữ tùy chỉnh.
